# คู่มือการทำงานปุ่ม Upgrade Structure (อัปเกรดโครงสร้างฐานข้อมูล)

คู่มือฉบับนี้อธิบายรายละเอียดการทำงานทีละขั้นตอนของปุ่ม **Upgrade Structure** ซึ่งถูกประมวลผลผ่านฟังก์ชัน `up_structure()` ใน [MainSettingController.php](file:///d:/Project%20Laravel/h-rims/app/Http/Controllers/Admin/MainSettingController.php) เพื่อปรับปรุงและจัดเตรียมฐานข้อมูลให้สอดคล้องกับประกาศเกณฑ์ราคาและโครงสร้างล่าสุด

---

## 📋 ลำดับขั้นตอนการทำงาน (Step-by-Step Flow)

### 1️⃣ การปรับปรุงตารางลูกหนี้การเงิน (Debtor Tables Upgrade)
* ระบบจะทำการตรวจสอบตารางกลุ่มลูกหนี้ทั้งหมด (ตารางที่ไม่มีคำว่า `_tracking` ลงท้าย เช่น `claim_op_ucs`, `claim_op_ofc` เป็นต้น)
* หากไม่พบคอลัมน์ที่จำเป็นสำหรับการเงินและการเคลม ระบบจะทำการเพิ่มคอลัมน์ต่อไปนี้เข้าสู่ตารางเป้าหมายโดยอัตโนมัติ:
  * `charge_date`, `charge_no`, `charge` (ราคาเรียกเก็บ)
  * `receive_date`, `receive_no`, `receive` (ราคาชำระจริง)
  * `repno` (เลขที่ใบเสร็จรับเงิน/เลข REP)
  * `adj_inc`, `adj_dec`, `adj_note`, `adj_date` (การปรับปรุงหนี้)
  * `debtor_change` (การเปลี่ยนแปลงลูกหนี้)

### 2️⃣ อัปเกรดระบบจัดเก็บข้อมูลส่งข้อมูล สปสช. (NHSO Endpoint Upgrade)
* ตรวจสอบและอัปเดตตาราง `nhso_endpoint` ทั้งฝั่งฐานข้อมูลระบบและฝั่ง HOSxP (`hrims.nhso_endpoint`)
* เพิ่มฟิลด์ควบคุมสถานะการดึง/ส่งข้อมูล (ถ้ายังไม่มี):
  * `claim_status` (สถานะการส่งเคลม)
  * `saved_at` (วันเวลาบันทึกรายการ)
  * `nhso_response` (ข้อความการตอบกลับจาก สปสช.)

### 3️⃣ อัปเกรดตารางรหัสรายการบริการของโรงพยาบาล (Hospital iCode)
* เพิ่มฟิลด์ **`sss_hc`** (High Cost ของสิทธิประกันสังคม) เข้าในตาราง `lookup_icode` พร้อมสร้างดัชนี (Index) เพื่อการสืบค้นที่รวดเร็วขึ้น

### 4️⃣ อัปเดตตารางและราคาเครื่องมือแพทย์สิทธิประกันสังคม (SSS Equipdev AIPN)
* **ลบตารางเก่า**: ดรอปตารางประกันสังคมรุ่นเก่า (`lookup_adp_sss`)
* **สร้างตารางใหม่**: สร้างตาราง `lookup_sss_equipdev_aipn` เพื่อรองรับโครงสร้างประกาศสิทธิประกันสังคมฉบับเต็ม
* **นำเข้าข้อมูล**: อ่านไฟล์ Excel จาก `docs/lookup/EquipdevAIPN.xlsx` เพื่อนำเข้ารายการอุปกรณ์และอัตราจ่ายชดเชยของประกันสังคม (อัตราที่ 1 `rate` และ อัตราที่ 2 `rate2`)

### 5️⃣ จัดการโครงสร้างบัญชียาโรงพยาบาลสำหรับสิทธิหลักประกันสุขภาพ (Drug Category CHI)
* ลบตารางหมวดหมู่ยารุ่นเก่า (`drugcat_aipn`)
* สร้างตาราง `drugcat_chi` เพื่อเตรียมรองรับบัญชีรหัสยามาตรฐานและรหัส TMT สำหรับการเบิกจ่ายสิทธิบัตรทอง

### 6️⃣ สร้างตารางจัดกลุ่มประเภทของรหัส ADP (NHSO ADP Type)
* ตรวจสอบและสร้างตาราง `lookup_nhso_adp_type`
* ทำการคัดลอกประเภทกลุ่ม ADP (เช่น ค่าห้อง/ค่าอาหาร, อุปกรณ์เครื่องมือแพทย์ ฯลฯ) มาจากตารางระบบของ HOSxP (`nhso_adp_type`)

### 7️⃣ รีเซ็ตและจัดทำตารางเกณฑ์ราคาเปรียบเทียบรหัส ADP (NHSO ADP Code Setup)
* ทำการดรอปและสร้างตารางหลัก **`lookup_nhso_adp_code`** ใหม่เพื่อปรับแต่งโครงสร้างฟิลด์สำหรับเกณฑ์เปรียบเทียบในแต่ละสิทธิ ได้แก่:
  * ราคาเปรียบเทียบสิทธิต่างๆ (`price_ucs`, `price_ofc`, `price_sss`, `price_lgo`, `price_fs`, `price_ucep`)
  * ฟิลด์สถานะมาร์กเกณฑ์ประกาศ: **`ins_ucs`** (เครื่องมือแพทย์สิทธิบัตรทอง) และ **`ins_ofc`** (เครื่องมือแพทย์สิทธิข้าราชการ)

### 8️⃣ นำเข้าเกณฑ์ราคาหลักและเชื่อมโยงราคาประกันสังคม (Main Excel & SSS Mapping)
* **8.1 นำเข้าข้อมูลเกณฑ์ราคาจาก Excel**: 
  อ่านข้อมูลจากไฟล์ Excel ตัวหลัก `docs/lookup/lookup_nhso_adp_code.xlsx` เข้าสู่ระบบทั้งหมด ซึ่งขั้นตอนนี้นอกจากจะได้ราคามาตรฐานของสิทธิต่างๆ แล้ว จะได้สถานะมาร์กว่ารายการใดเป็นเครื่องมือแพทย์ตรงตามเกณฑ์ประกาศฉบับล่าสุดทันที:
  * รายการเครื่องมือแพทย์ของ UCS: มาร์กสถานะ `ins_ucs = 'Y'` (142 รายการ)
  * รายการเครื่องมือแพทย์ของ OFC: มาร์กสถานะ `ins_ofc = 'Y'` (536 รายการ)
* **8.2 อัปเดตราคาประกันสังคม (SSS Price Connection)**:
  ทำการเชื่อมโยงข้อมูลข้ามตาราง โดยนำอัตราจ่ายล่าสุด (`rate`) จากตารางสิทธิประกันสังคม `lookup_sss_equipdev_aipn` (ข้อมูลในขั้นตอนที่ 4) ที่มีผลบังคับใช้ในปัจจุบัน (`dateexp >= DATE(NOW())`) มาอัปเดตใส่เข้าช่อง `price_sss` ของตารางเกณฑ์ราคาหลัก และใช้ตาราง `lookup_sss_equipdev_aipn` นี้เป็นแหล่งดึงข้อมูลราคาสิทธิ์ประกันสังคมโดยตรงเพื่อแสดงผลในหน้าตารางเปรียบเทียบรหัส (Lookup iCode) ของสิทธิ์ประกันสังคม (SSS-HC)
