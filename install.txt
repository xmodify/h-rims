1.Clone the repository (first time only)
	git clone https://github.com/xmodify/h-rims.git
	cd h-rims
2.Pull latest changes (if already cloned)
	git pull origin main
3.Install PHP dependencies
	composer install
	npm install
	npm run dev
	npm run build
4.Copy the .env file
	cp .env.example .env
5.Generate the application key
	php artisan key:generate
6.Run migrations (if needed)
	php artisan migrate
7.Serve the project
	php artisan serve

-----------------------------------------------------------------------
1.Initialize Git (if not already done)
	git init
2.Create a .gitignore file
	/vendor
	/node_modules
	.env
	/storage/*.key
	/public/storage
3.Add your files to Git
	git add .
4.Commit the files
	git commit -m "Initial commit of h-rims"
5.Add remote repository
	git remote add origin https://github.com/xmodify/h-rims
6.Push the code
	git branch -M main
	git push -u origin main
